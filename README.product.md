# Tarot Psychic Reader

## What it is

A presentable shipping product folder for the Tarot Psychic Reader project.

## Status

Shipping local-web MVP available as **v1.1.0**.

## Current shipping build

- `app/local-web/` — local web MVP with ritual intake, spread selection, reading output, and local browser history
- `app/prototype/` — earlier prototype kept for reference and shared local image assets

## Structure

- `app/` — product files / implementation
- `assets/` — images, brand assets, media
- `docs/` — polished product docs
