# Tarot Reader v2.1.0

Public code export for the Tarot Reader shipping product.

## Included
- local web app frontend
- Python backend
- tarot card assets currently included in the shipping product
- OpenAI-backed two-call reading flow

## Not included
- real API keys
- local workspace memory/history
- internal factory notes

## Run locally

```bash
cd app/local-web
export OPENAI_API_KEY=your_key_here
npm start
```

Then open:

```text
http://127.0.0.1:4173
```
